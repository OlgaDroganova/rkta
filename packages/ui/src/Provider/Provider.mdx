# Provider

Use Provider to wrap your application and create `@rkta/ui` context.

## Basic usage

### Props

<Props of={Provider} />

### Example

```javascript
import Provider as RktaProvider from '@rkta/ui/Provider';

<RktaProvider theme={{ color: { customColor: '#fff' } }}>
...
</RktaProvider>
```

## Context Hook

```javascript
import { useProviderContext } from '@rkta/ui/Provider';

function MyComponent() {
  const { theme, replaceTheme } = useProviderContext();
  ...
}
```
